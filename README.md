# Archana's DSA Pattern Handbook (JavaScript)

A teaching-style guide to mastering 15 essential DSA patterns for coding interviews.

Each pattern includes:
- Concept explanation
- Real-world analogy
- Visual diagrams
- JavaScript template
- When to use
- Common mistakes
- Practice problems

Goal: Pattern recognition over memorization.

---

## Patterns Included

1. Prefix Sum
2. Two Pointers
3. Sliding Window
4. Fast & Slow Pointers
5. In-place Reversal
6. Monotonic Stack
7. Top K Elements
8. Intervals
9. Modified Binary Search
10. Tree Traversals
11. DFS
12. BFS
13. Matrix Traversal
14. Backtracking
15. Dynamic Programming

Write the template from memory.
